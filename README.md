# STM32-Industrial-Gateway (基于STM32的工业/物联网通用控制网关)

本项目是一个基于 STM32F103/F407 芯片开发的通用嵌入式控制网关。项目实现了完整的硬件驱动封装、多路传感器数据采集、低功耗电源管理以及工业常用的 Modbus/MQTT 通信协议。

项目代码完全模块化设计，无任何臃肿第三方库依赖，非常适合作为中小型工业控制、智能家居项目的底层脚手架。

## 核心功能模块 (Features)

* **多路通信协议栈集成**：
  * **Modbus-RTU 协议**：手写解析状态机，支持多路 RS485 从机数据采集与寄存器读写。
  * **MQTT 协议网关**：通过 USART 驱动 ESP8266/4G 模块，实现 AT 指令解析及上云（支持阿里云/腾讯云）。
* **传感器与外设驱动库 (Bare-Metal / HAL)**：
  * 完整实现了 I2C (EEPROM AT24C02 / 温湿度 SHT30)、SPI (Flash W25Q64)、ADC (多通道DMA采集 + 软件一阶滤波) 的稳定驱动。
* **低功耗与电源管理**：
  * 支持 ST 芯片的 Sleep / Stop 模式切换，通过外部中断（RTC/GPIO）定时唤醒，整机待机功耗控制在微安（uA）级别。
* **设备参数参数配置系统**：
  * 利用 Flash 模拟 EEPROM，实现设备掉电参数保存（如 Wi-Fi 密码、设备 ID、校准参数）。

##  项目硬件拓扑结构

传感器 (I2C/SPI/ADC) ---> STM32 主控 ---> RS485 现场总线 (Modbus-RTU)
                             |
                             +---> 4G / Wi-Fi 模块 (AT指令) ---> 云端大屏
